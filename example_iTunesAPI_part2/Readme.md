# AppStoreClone
iTunes API를 활용한 앱스토어 검색 탭 구현

***
### 앱 소개
API통신과 레이아웃을 이해하며, 애플의 HIG를 보며 심미성과 사용친화적으로 표현된 AppStore를 클론하였습니다.

***
### 목표
1. iTunesAPI를 활용한 AppStore 클론
2. AutoLayout 설정

***
### 활용 기술 
1. CollectionView 
2. StoryBoard Layout
3. Network
